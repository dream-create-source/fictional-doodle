# Token Transformer Project 3 Snapshot

This directory preserves the larger tokenizer-based transformer iteration.

Notable architecture defaults from `common.py`:

- `embed_dim = 768`
- `num_heads = 12`
- `num_layers = 10`
- `max_seq_len = 256`
- `vocab_size = 8192`

This project is included as a token-based modeling iteration, separate from the root 38.3M-parameter character-level GPT checkpoints.
